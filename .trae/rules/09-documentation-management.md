## 9.2 文档规范

### README 结构

```markdown
# 项目名称

## 简介
项目简介和核心功能

## 快速开始
安装和运行步骤

## 功能特性
主要功能列表

## 配置说明
配置项说明

## API文档
接口文档链接

## 常见问题
FAQ

## 贡献指南
如何参与开发

## 许可证
开源协议
```

### API文档规范

```markdown
## 接口名称

### 请求
- 方法: POST
- 路径: /api/endpoint
- 认证: 需要

### 参数
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| param1 | string | 是 | 参数说明 |

### 请求示例
```json
{
    "param1": "value1"
}
```

### 响应
| 字段名 | 类型 | 说明 |
|--------|------|------|
| code | int | 状态码 |
| data | object | 数据 |

### 响应示例
```json
{
    "code": 200,
    "data": {}
}
```

### 错误码
| 错误码 | 说明 |
|--------|------|
| 400 | 参数错误 |
| 401 | 未认证 |
```
