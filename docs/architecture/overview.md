## 技术选型

### 后端技术

| 技术 | 版本 | 选型理由 |
|------|------|----------|
| FastAPI | 0.115.0+ | 高性能异步框架，自动 API 文档 |
| Uvicorn | 0.34.0+ | ASGI 服务器，支持 WebSocket |
| SQLite | 3.x | 轻量级数据库，无需独立服务 |
| Pydantic | 2.10.0+ | 数据验证，类型安全 |
| loguru | - | 结构化日志，按日期分割 |

### 前端技术

| 技术 | 版本 | 选型理由 |
|------|------|----------|
| Bootstrap | 5.x | 响应式 UI 框架 |
| Vanilla JS | ES6+ | 轻量级，无构建依赖 |

### 外部集成

| 服务 | 用途 | 协议 |
|------|------|------|
| 闲鱼 WebSocket | 实时消息 | WebSocket |
| 闲鱼 API | 商品/订单操作 | HTTPS |
| OpenAI API | 智能回复 | HTTPS |

## 核心流程

### 消息处理流程

```
1. WebSocket 接收消息
2. 解析消息类型和内容
3. 查询关键词匹配规则
4. AI 智能回复（可选）
5. 发送回复消息
6. 记录操作日志
```

### 自动发货流程

```
1. 检测订单状态变更
2. 匹配发货规则
3. 获取虚拟商品内容
4. 调用闲鱼发货 API
5. 更新订单状态
6. 通知用户
```

### 数据流向

```
用户请求 → Nginx → FastAPI → 业务逻辑 → 数据库
                              ↓
                         外部服务（闲鱼/OpenAI）
```

## 安全设计

### 认证机制

- JWT Token 认证
- Token 有效期：24 小时
- 刷新机制：滑动过期

### 数据隔离

- 用户级别数据隔离
- Cookie 加密存储
- 敏感信息环境变量

### 请求限流

- 全局限流：100 请求/分钟
- 登录限流：5 次/分钟
- API 限流：根据接口配置

## 扩展性设计

### 水平扩展

- 无状态 API 设计
- 数据库可迁移至 PostgreSQL
- 支持多实例部署

### 功能扩展

- 插件化消息处理器
- 可配置的回复策略
- 模块化的发货规则
