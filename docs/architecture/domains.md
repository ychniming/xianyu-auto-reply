## 领域详情

### 用户域 (User Domain)

**职责：** 用户认证、权限管理、操作日志

| 实体 | 说明 |
|------|------|
| User | 用户账号信息 |
| Role | 用户角色 |
| OperationLog | 操作日志 |

**核心规则：**
- 用户名唯一
- 密码加密存储
- 操作可追溯

### 账号域 (Account Domain)

**职责：** 闲鱼账号管理、Cookie 维护、状态监控

| 实体 | 说明 |
|------|------|
| Cookie | 闲鱼账号凭证 |
| AccountStatus | 账号状态 |
| TokenInfo | Token 信息 |

**核心规则：**
- Cookie 有效期约 7 天
- 自动刷新 Token
- 异常状态告警

### 消息域 (Message Domain)

**职责：** 消息收发、关键词匹配、自动回复

| 实体 | 说明 |
|------|------|
| Message | 消息内容 |
| Keyword | 关键词规则 |
| ReplyRule | 回复规则 |

**核心规则：**
- 实时消息处理
- 关键词优先匹配
- 支持多种消息类型

### 商品域 (Product Domain)

**职责：** 商品管理、库存维护、价格调整

| 实体 | 说明 |
|------|------|
| Product | 商品信息 |
| Inventory | 库存信息 |
| PriceRule | 价格规则 |

**核心规则：**
- 商品状态同步
- 库存自动扣减
- 价格策略执行

### 订单域 (Order Domain)

**职责：** 订单处理、状态追踪、异常处理

| 实体 | 说明 |
|------|------|
| Order | 订单信息 |
| OrderStatus | 订单状态 |
| OrderLog | 订单日志 |

**核心规则：**
- 状态机流转
- 异常订单标记
- 自动确认机制

### 发货域 (Delivery Domain)

**职责：** 自动发货、虚拟商品管理、物流对接

| 实体 | 说明 |
|------|------|
| DeliveryRule | 发货规则 |
| VirtualGoods | 虚拟商品 |
| DeliveryLog | 发货日志 |

**核心规则：**
- 规则匹配发货
- 虚拟商品自动发送
- 发货失败重试

### AI回复域 (AIReply Domain)

**职责：** 智能回复、意图识别、上下文管理

| 实体 | 说明 |
|------|------|
| Conversation | 对话上下文 |
| Intent | 用户意图 |
| AIConfig | AI 配置 |

**核心规则：**
- 上下文窗口管理
- 意图分类
- 回复质量控制

### 系统域 (System Domain)

**职责：** 配置管理、日志记录、健康检查

| 实体 | 说明 |
|------|------|
| Config | 系统配置 |
| SystemLog | 系统日志 |
| HealthStatus | 健康状态 |

**核心规则：**
- 配置热更新
- 日志分级存储
- 健康状态监控

## 领域交互

```
用户域 ──认证──> 账号域 ──管理──> 消息域
                  │                │
                  └──监控──> 商品域 <──关联──┘
                                │
                                ▼
                            订单域 ──触发──> 发货域
                                │
                                └──辅助──> AI回复域
```
