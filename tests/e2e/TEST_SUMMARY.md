# 闲鱼自动回复系统 - 端到端测试总结报告

## 测试概述

本测试套件为闲鱼自动回复系统提供了全面的点击操作端到端测试，使用 Playwright 测试框架实现，覆盖关键用户交互路径。

## 运行测试

### 安装依赖
```bash
cd tests/e2e
npm install
npx playwright install
```

### 运行所有测试
```bash
npm test
```

### 运行特定测试
```bash
npm run test:login      # 登录测试
npm run test:navigation # 导航测试
npm run test:accounts   # 账号管理测试
npm run test:keywords   # 关键词管理测试
```

### 带界面运行
```bash
npm run test:headed
```

### UI模式
```bash
npm run test:ui
```

### 查看报告
```bash
npm run report
```
